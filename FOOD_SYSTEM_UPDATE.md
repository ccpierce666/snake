# 食物系统更新 - 等级与稀有度重设

## 更新日期
2026年2月25日

## 核心变更

从旧的 **8等级系统（value 1-8）** 升级到新的 **11等级系统（value 1-11）**

### 旧系统 → 新系统对应

| 旧等级 | 旧基础增长 | 新等级 | 新基础增长 | 概率变化 |
|-------|---------|-------|---------|--------|
| 1 | 3 | 1 | 2 | 50% |
| 1 | 3 | 2 | 4 | 15% |
| 2 | 6 | 3 | 8 | 13% |
| 2 | 6 | 4 | 16 | 10% |
| 3 | 12 | 5 | 32 | 6% |
| - | - | 6 | 64 | 3.5% ⭐**新增高等级** |
| 4 | 24 | 7 | 128 | 1.7% |
| 5 | 56 | 8 | 256 | 0.6% |
| 6 | 224 | 9 | 512 | 0.15% |
| 7 | 512 | 10 | 1024 | 0.04% |
| 8 | 1024 | 11 | 2048 | 0.01% |

## 详细改动清单

### 1. 服务器端 - `SnakeGameService.lua`

#### L126-145：食物生成函数 `spawnFood()`
**改动：** 重写概率递减表和等级分配

**旧方式（8等级）:**
```lua
if rand < 0.001 then value = 8
elseif rand < 0.003 then value = 7
-- ... 等等
else value = 1 end
```

**新方式（11等级）:**
```lua
if rand < 0.0001 then value = 11       -- 0.01%
elseif rand < 0.0005 then value = 10   -- 0.04%
elseif rand < 0.002 then value = 9     -- 0.15%
-- ... 等等
else value = 1 end                     -- 50%
```

**关键变更：**
- 增加了等级6-11（6个新等级）
- 高等级食物概率大幅降低（等级6+概率<4%）
- 基础等级1的概率从57%提升到50%

#### L306-320：增长映射表
**改动：** 从旧的value 1-8改为新的value 1-11，基础增长改为2的倍数

```lua
local growthMap = {
    [1] = 2,       -- 2的倍数序列
    [2] = 4,
    [3] = 8,
    -- ... 直到
    [11] = 2048
}
```

### 2. 客户端 - `SnakeGame3DView.lua`

#### L529-568：食物颜色与大小渲染
**改动：** 完整重写食物的外观逻辑

**颜色渐进体系：** 绿→黄→橙→红→蓝→紫

```lua
if value == 11 then
    size = 4.2
    color = Color3.fromRGB(255, 0, 255) -- 紫红（最珍稀）
elseif value == 10 then
    size = 4.0
    color = Color3.fromRGB(200, 0, 255) -- 紫色
-- ... 继续渐进
else
    size = 1.0
    color = Color3.fromRGB(100, 255, 100) -- 绿色（最常见）
end
```

**大小递增：** 1.0 → 4.2（与等级成正比）

### 3. 客户端 - `SnakeGameController.lua`

#### L118-128：增长映射表
**改动：** 与服务器同步，添加等级6-11的映射

#### L140-141：基础增长计算
**改动：** 从 `diff * 3` 改为 `diff * 2`

```lua
-- 旧：基于等级1的基础增长3
local baseGrowth = diff * 3

-- 新：基于等级1的基础增长2
local baseGrowth = diff * 2
```

## 新系统特性

### ✅ 视觉区分清晰
- 每个等级都有独特的颜色
- 大小从1.0到4.2，直观体现等级
- 玩家能立即识别食物的价值

### ✅ 概率分布合理
```
等级1-5（常见）   : 94% 的概率
等级6（稀有）     : 3.5% 的概率  ← **64以后概率很小**
等级7-11（极稀有）: 2.5% 的概率
```

### ✅ 成长阶梯清晰
- 基础增长: 2, 4, 8, 16, 32, 64, 128, 256, 512, 1024, 2048
- 完美的2的倍数序列，易于理解和预测
- 每个等级的价值倍增

### ✅ 游戏寿命延长
- 高等级食物极其稀有
- 即使玩了很长时间，也可能只见过几个紫色食物
- 维持新鲜感和惊喜感

## 等级详解

### 绿色区域（等级1-2）
- **常见程度**：65%
- **增长**：+2 到 +4
- **用途**：初期积累
- **特征**：随处可见，几乎不值一提

### 黄色区域（等级3-4）
- **常见程度**：23%
- **增长**：+8 到 +16
- **用途**：早期加速
- **特征**：逐步增多，有目标感

### 橙红区域（等级5-6）
- **常见程度**：9.5%
- **增长**：+32 到 +64
- **用途**：中期提升
- **特征**：开始显得珍贵，**转折点**

### 蓝紫区域（等级7+）
- **常见程度**：2.5%
- **增长**：+128 及以上
- **用途**：后期突破
- **特征**：极端稀有，发现时有惊喜

降低阈值：
```lua
elseif rand < 0.12 then value = 6  -- 从0.06改为0.12，概率翻倍
```

### 基础增长2太慢？
使用倍数缓冲（配合非线性增长系统）：
```lua
local scaledGrowth = math.floor(baseGrowth * growthMultiplier * 1.5 + 0.5)
```
