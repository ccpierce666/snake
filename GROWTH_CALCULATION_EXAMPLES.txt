  非线性增长倍数系统 - 实际计算示例
═══════════════════════════════════════════════════════════════

【公式】
倍数 = 1.0 + (currentLength / 100000)^0.5
最终增长 = floor(基础增长 × 倍数 + 0.5)

═══════════════════════════════════════════════════════════════
示例1：早期阶段 - 蛇长50段，吃了value=1的食物
───────────────────────────────────────────────────────────────
当前长度：50
倍数 = 1.0 + (50/100000)^0.5 = 1.0 + (0.0005)^0.5 = 1.0 + 0.0224 = 1.0224
基础增长：3
实际增长 = floor(3 × 1.0224 + 0.5) = floor(3.567) = 3 或 4
结果：蛇增长 3-4 段（几乎没有加成）

示例2：中期阶段 - 蛇长5000段，吃了value=3的食物
───────────────────────────────────────────────────────────────
当前长度：5000
倍数 = 1.0 + (5000/100000)^0.5 = 1.0 + (0.05)^0.5 = 1.0 + 0.2236 = 1.2236
基础增长：12
实际增长 = floor(12 × 1.2236 + 0.5) = floor(14.283) = 14
结果：蛇增长 14 段（有 22% 的加成）

示例3：中后期 - 蛇长50000段，吃了value=5的食物
───────────────────────────────────────────────────────────────
当前长度：50000
倍数 = 1.0 + (50000/100000)^0.5 = 1.0 + (0.5)^0.5 = 1.0 + 0.7071 = 1.7071
基础增长：56
实际增长 = floor(56 × 1.7071 + 0.5) = floor(95.898) = 96
结果：蛇增长 96 段（有 71% 的加成）

示例4：后期阶段 - 蛇长100000段，吃了value=8的食物
───────────────────────────────────────────────────────────────
当前长度：100000
倍数 = 1.0 + (100000/100000)^0.5 = 1.0 + 1.0 = 2.0
基础增长：1024
实际增长 = floor(1024 × 2.0 + 0.5) = floor(2048.5) = 2048
结果：蛇增长 2048 段（有 100% 的加成，双倍增长）

示例5：超大蛇 - 蛇长200000段（超过上限），吃了value=4的食物
───────────────────────────────────────────────────────────────
当前长度：200000
倍数计算：min(200000, 100000) = 100000
倍数 = 1.0 + (100000/100000)^0.5 = 1.0 + 1.0 = 2.0（已达上限）
基础增长：24
实际增长 = floor(24 × 2.0 + 0.5) = floor(48.5) = 48
结果：蛇增长 48 段（始终保持 2.0 倍的最大加成）

═══════════════════════════════════════════════════════════════
批量吃食物的例子 - 蛇长20000段，一次吃5个食物
───────────────────────────────────────────────────────────────
当前长度：20000
倍数 = 1.0 + (20000/100000)^0.5 = 1.0 + 0.4472 = 1.4472

食物1（value=1）：基础增长3，实际 = floor(3 × 1.4472) = 4
食物2（value=2）：基础增长6，实际 = floor(6 × 1.4472) = 8
食物3（value=2）：基础增长6，实际 = floor(6 × 1.4472) = 8
食物4（value=3）：基础增长12，实际 = floor(12 × 1.4472) = 17
食物5（value=4）：基础增长24，实际 = floor(24 × 1.4472) = 34

总增长 = 4 + 8 + 8 + 17 + 34 = 71 段

核心设计理念
═══════════════════════════════════════════════════════════════

✓ 早期保留挑战：0-1000段阶段倍数接近1.0，增长最慢
  → 新手需要花时间学习和积累，不会太快变大

✓ 中期加速节奏：1000-100000段阶段倍数线性上升
  → 随着进度而加快，维持持续的成就感

✓ 后期稳定倍数：100000+段阶段倍数固定2.0
  → 防止蛇变得无限大，保持游戏平衡

✓ 非线性而非线性：使用平方根而非直线
  → 早期增长缓慢，中期加速，后期趋缓，自然流畅
