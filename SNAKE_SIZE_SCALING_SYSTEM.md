# 蛇身体动态大小缩放系统

## 系统概述

用户反馈：早期蛇太大，后期应该逐步放大。新系统采用**动态计算**而不是固定的等级分层，让蛇的大小与身体长度形成完美的视觉对应关系。

## 设计理念

```
早期（个位数）      中期（几k-几十k）      后期（几百k-10m+）
  极小蛇             中等蛇                大蛇
 1/3格子            1格左右              1.0-2.5+
  0.33              0.7-1.0              1.0-2.5
```

## 代码实现

### 动态大小计算函数

```lua
local function calculateBodySize(currentLength)
    if currentLength < 10 then
        return 0.33
    elseif currentLength < 100 then
        local ratio = (currentLength - 10) / 90
        return 0.5 + ratio * 0.2
    elseif currentLength < 10000 then
        local ratio = (currentLength - 100) / 9900
        return 0.7 + ratio * 0.3
    elseif currentLength < 100000 then
        local ratio = (currentLength - 10000) / 90000
        return 1.0 + ratio * 0.5
    elseif currentLength < 1000000 then
        local ratio = (currentLength - 100000) / 900000
        return 1.5 + ratio * 1.0
    else
        return 2.5
    end
end
```

### 计算逻辑说明

每个分段都使用**线性插值**：
```
新大小 = 起始大小 + (当前位置 / 分段长度) × 增长量
```

**例子：** 身体长度5000时，属于第3阶段（100-10000）
```
ratio = (5000 - 100) / 9900 = 0.4949
size = 0.7 + 0.4949 × 0.3 = 0.7 + 0.1485 = 0.8485
```

| 分界点 | 原因 | 对应游戏进度 |
|-------|------|-----------|

✅ **无性能损失**
