## 🐛 遇到问题?

### 问题1: Output显示"找不到hill模型"
**解决**: 这是正常的! 代码会自动使用默认围墙。  
**改进**: 按照方案1或2创建Hill模型。

### 问题2: 脚本运行后still找不到Hill
**检查**:
1. 模型名字大小写? 必须是 `hill` 小写
2. 位置对吗? 必须在 Workspace 根目录
3. 类型对吗? 必须是 Model 或 Part

```lua
-- 在Command Bar运行此代码验证:
print(workspace:FindFirstChild("hill"))
-- 应该输出模型信息，不是nil
```

### 问题3: 默认围墙太丑
**解决**: 编辑RGB颜色值
```lua
-- 找到这行:
local wallColor = Color3.fromRGB(120, 200, 100)

-- 改成你喜欢的颜色:
-- 更亮的绿色: Color3.fromRGB(150, 220, 120)
-- 蓝色: Color3.fromRGB(100, 150, 200)
-- 紫色: Color3.fromRGB(180, 100, 200)
```
